# Proyecto-Modular-Retinopat-a
Caracterización cuantitativa de la arquitectura vascular retiniana mediante dimensión fractal y lacunaridad para clasificar estadios de retinopatía diabética. Implementado en MATLAB con segmentación por filtro de Frangi y validación mediante curvas ROC/AUC sobre 2,495 imágenes de fondo de ojo.
